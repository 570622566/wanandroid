# wanandroid
## 玩安卓微信小程序

玩安卓微信小程序是基于 [玩安卓api](https://wanandroid.com/index) 开发的一款比较完整，UI质量比较高的项目，值得新手学习。

## 能从该项目中学到什么？

主要功能点
1. 自定义标题栏
2. tabs + swiper 联动
3. 双scroll-view 联动 
4. 自定义component及数据传递，事件传递
